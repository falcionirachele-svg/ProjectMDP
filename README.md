# 📌 Project Bivio
Videogioco di ruolo testuale interattivo basato su bivi narrativi. 
Il gioco permette all'utente di esplorare una storia dinamica basata su dialoghi, combattimenti e indovinelli.

## 🚀 Come eseguire il progetto

### Prerequisiti
- Java 25 (LTS)
- Gradle

### Istruzioni
git clone https://github.com/falcionirachele-svg/ProjectMDP.git
cd ProjectBivio

### Build del progetto
./gradlew build

### Esecuzione
./gradlew run

Una volta avviato si aprirà automaticamente la finestra di gioco e vi saranno tre possibili azioni da parte dell'utente:
* Bivio Narrativo: verrà mostrato il testo della storia con una richiesta, l'utente potrà accettare o rifiutare
* Bivio Indovinello: verrà posta un tranello sullo schermo con 4 possibilità, fino a che l'utente non indovina non potrà proseguire
* Bivio Combattimento: l'utente prima e in qualsiasi momento durante lo scontro potrà scegliere se continuare o fuggire. Per ogni turno verranno illustrati i risultati dello scontro.


## 🤖 Uso di strumenti di AI

* Utilizzato Copilot per:

  * aiuto legato alle configurazioni dell'ambiente
  * chiarire errori di compilazione
  * comprendere concetti teorici
